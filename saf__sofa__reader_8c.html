Detailed Description

Public source for the sofa reader module (SAF_SOFA_READER_MODULE)

Note
This (optional) SOFA reader, requires either netcdf (and its dependencies) or zlib to be linked to your project. Note that zlib is used by default, unless "SAF_ENABLE_NETCDF" is defined. Refer to docs/SOFA_READER_MODULE_DEPENDENCIES.md for more information.
